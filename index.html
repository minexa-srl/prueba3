Introducción a las Redes

¿Qué son las redes de computadoras?

Las redes de computadoras son sistemas fundamentales de comunicación que permiten la interconexión de múltiples dispositivos (como ordenadores, servidores, y dispositivos inteligentes) con el fin primordial de compartir información, recursos y servicios.

Importancia en el mundo moderno

Gracias a las redes, hoy es posible enviar mensajes instantáneos, realizar videollamadas, acceder a grandes volúmenes de información en tiempo real y trabajar de forma colaborativa desde cualquier lugar del mundo. Son la columna vertebral de la sociedad de la información.

Modelos de Referencia: OSI y TCP/IP

El Modelo OSI (7 Capas)

El modelo OSI (Open Systems Interconnection) es un estándar de 7 capas que ayuda a entender el proceso de comunicación desde el hardware físico hasta la aplicación del usuario. Provee una estructura teórica para la interoperabilidad.

El Modelo TCP/IP (4 Capas)

Por otro lado, el modelo TCP/IP es el protocolo pragmático utilizado en Internet, simplificando el proceso en 4 capas clave. Es el modelo funcional que hace posible la red global actual.

Protocolos de Comunicación

¿Qué es un protocolo?

Los protocolos son el "lenguaje" de la red. Son conjuntos de reglas y estándares que permiten que dispositivos heterogéneos se comuniquen correctamente. Sin ellos, el intercambio de datos sería caótico.

Protocolos Fundamentales

Algunos protocolos esenciales que utilizamos diariamente son:

  • HTTP: Para la navegación web.
  • IP: Para el direccionamiento y enrutamiento.
  • TCP: Para asegurar transmisiones fiables.
  • DNS: Para traducir nombres de dominio.

Topologías de Red

Disposición Física y Lógica

La topología de red define la disposición física o lógica de los dispositivos y sus conexiones. La elección de la topología afecta la escalabilidad, el coste y la facilidad de administración.

Topologías Comunes y Características

Diferentes tipos de topologías y sus usos:

  • Estrella: (Común) Centralizada en un switch. Fácil de diagnosticar.
  • Malla: (Reliable) Redundante. Alta fiabilidad, pero costosa.
  • Bus: (Simple) Un solo cable troncal.

Software de Red y Simuladores

Software de Administración y Monitoreo

El software de red es crucial para la administración, monitoreo y seguridad. Incluye desde los sistemas operativos de red hasta herramientas de diagnóstico de tráfico y gestión de seguridad.

Simuladores como Herramientas Educativas

Simuladores como Cisco Packet Tracer: Son herramientas educativas potentes que permiten a estudiantes y profesionales diseñar, configurar y solucionar problemas de redes en un entorno virtual seguro, sin necesidad de hardware físico costoso.

Dispositivos de Red Esenciales

Hardware de Interconexión

Para construir una red funcional, se requieren dispositivos de hardware que gestionen el flujo de datos y conecten los hosts de manera eficiente.

Principales Componentes y Funciones

  • Router: Conecta redes distintas y determina la mejor ruta.
  • Switch: Conecta dispositivos dentro de una misma LAN (inteligente).
  • Access Point: Permite la conexión inalámbrica Wi-Fi.

Cableado UTP y Códigos de Colores

El Medio Físico Estándar

El cable UTP (Unshielded Twisted Pair) es el estándar físico más utilizado para redes LAN cableadas. Su rendimiento depende de una terminación correcta y precisa en los conectores RJ-45.

Los Estándares T568A y T568B

Los estándares T568A y T568B definen el orden preciso de los 8 hilos de colores para garantizar la compatibilidad y el rendimiento. El orden T568B es el más extendido universalmente.
